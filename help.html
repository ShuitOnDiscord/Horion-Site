what are horion commands

Commands are like the vanilla slash commands, where they are ran in the chat. Horion commands, however must start with a period (.) instead of a slash (/). These commands can range from ejecting horion from minecraft to setting up a module to giving you items without the required permissions

what are horion modules

Modules can be described as the options in categories. The modules are organized within the categories that describe it. For example, the zoom module lets you see far objects closely, so it would be in the visual category

default keybinds

default keybinds can be found and edited in the config file space%localappdata%\Packages\Microsoft.MinecraftUWP_8wekyb3d8bbwe\RoamingState\Horion\default.h . they are stored as decimal values, which can be found by searching it on any search engine. in the config file, you can also easily see what modules are toggled. this can be useful for disabling a module that is crashing your game.
